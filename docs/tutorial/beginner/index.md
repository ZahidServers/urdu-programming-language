## What You Will Learn — کیا سیکھیں گے

By the end of this section you will be able to:

- Write and run Urdu PL programs
- Store and use data in variables
- Read input from the user and display output
- Work with text, numbers, and booleans
- Make decisions with `اگر`/`ورنہ`
- Repeat actions with `جبکہ` and `کے_لیے` loops
- Organize code into reusable functions

> **اردو:** اس سیکشن کے بعد آپ متغیر، ڈیٹا اقسام، ان پٹ/آؤٹ پٹ، شرطی بیانات، لوپ، اور فنکشنز کے ساتھ اصل پروگرام بنا سکیں گے۔
